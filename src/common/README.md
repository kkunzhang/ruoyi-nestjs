# Common 层（通用能力层）

这里存放通用功能模块。

## 目录结构
```
common/
├── prisma/           # Prisma 数据库连接
├── pagination/       # 分页功能
├── auth/             # 权限认证（JWT、角色权限）
├── logger/           # 日志系统
├── filters/          # 全局异常过滤器
├── interceptors/     # 拦截器（日志、响应格式化）
├── decorators/       # 自定义装饰器
├── guards/           # 守卫（权限校验）
├── pipes/            # 管道（数据转换、验证）
└── utils/            # 工具函数
```

## 功能说明
- **分页**：统一分页参数和响应格式
- **权限**：JWT 认证 + 角色权限控制
- **日志**：操作日志、登录日志、系统日志
- **异常处理**：统一异常响应格式
- **响应格式化**：统一 API 响应结构
